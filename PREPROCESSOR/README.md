#cpp-preprocessor
## "Препроцессор"
Программа, которая проводит препроцессинг кода - заменяет указания билиотек #include <...> или "..." на их код. 

### Системные требования.

C++ 17 стандарт.

### Работа программы

Программа тестирует препроцессор на тестах в main().
